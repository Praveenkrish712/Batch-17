3) Types of selectors
==> elements selector (h1) (h2) (p)
==> Class selector - (.) - No of times - Multiple times we can use the same class name
==> Id selector - (#) - unique - only one we can use the same id name
==> Universal selector - (*) - it will apply the style into the whole site
==> Group selector - class or id selector contains different tags we wanna change the same style means we go with Group selector

8) Flex (display) - one dimensional layout
==> using flex to parent it will accore column into row
i) Flex direction - row, column, column-reverse, row-reverse
ii) justify content(X) - start, center, end, space-around, space-evenly, space-between
iii) Align items(Y) (set the height then only align item will work) - start, center, end
iv) gap - to be assgin with only values 
v) flex-wrap - no-wrap, wrap, wrap-reverse
vi) Align content (Y)(set the height) (only flex wrap is enable then only it will work) - start, center, end, space-around, space-evenly, space-between
vii) flex (on item) to be increase the empty space(width)
viii) order - change the visual order of the property
